# 📊 Customer Personal Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents a comprehensive **Customer Personal Analysis Dashboard** built using Power BI.
The goal is to analyze customer demographics, spending behavior, and purchasing patterns to derive actionable business insights.

---

## 🎯 Objectives

* Understand customer spending habits across product categories
* Identify high-value customers based on income and purchases
* Analyze customer demographics (age, education, marital status)
* Provide insights for targeted marketing strategies

---

## 📂 Dataset Information

The dataset contains customer-level information including:

* **Demographics**: Year_Birth, Education, Marital_Status
* **Financials**: Income
* **Family Info**: Kidhome, Teenhome
* **Customer Behavior**: Recency, NumWebPurchases, NumStorePurchases
* **Product Spending**:

  * MntWines
  * MntFruits
  * MntMeatProducts
  * MntFishProducts
  * MntSweetProducts
  * MntGoldProds

---

## 🛠️ Data Preparation

* Cleaned missing and inconsistent values
* Converted data types (e.g., Income to numeric)
* Created new columns such as:

  * **Age** (from Year_Birth)
  * **Total Spending** (sum of all product categories)
* Used **Unpivot transformation** for product-wise analysis

---

## 📊 Dashboard Features

### 🔹 1. Customer Demographics

* Age distribution
* Education level analysis
* Marital status breakdown

### 🔹 2. Income Analysis

* Income distribution across customers
* Relationship between income and spending

### 🔹 3. Product-wise Spending

* Bar chart showing spending across categories
* Identification of top-performing products

### 🔹 4. Purchase Behavior

* Web vs Store vs Catalog purchases
* Monthly website visits

### 🔹 5. Customer Segmentation

* High-value vs low-value customers
* Recency-based segmentation

---

## 📈 Key Insights

* Wine and meat products contribute the highest revenue
* Higher income customers tend to spend more on premium products
* Customers with fewer recent purchases may require re-engagement strategies
* Web purchases are increasing compared to store purchases
